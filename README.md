# AMRC Daily Center

GitHub Pages 免費版每日內容中心。

預設網址：

```text
https://weisen83.github.io/amrc-daily/
```

## 上傳方式

1. 到 GitHub 建立新 Repository
2. Repository name 輸入：

```text
amrc-daily
```

3. 選 Public
4. 按 Create repository
5. 按 Upload files
6. 上傳本資料夾內所有檔案：
   - index.html
   - style.css
   - script.js
   - README.md
7. Commit changes

## 開啟 GitHub Pages

1. 進入 Repository
2. Settings
3. Pages
4. Source 選 Deploy from a branch
5. Branch 選 main
6. Folder 選 /root
7. Save

等待 1–3 分鐘後，打開：

```text
https://weisen83.github.io/amrc-daily/
```

## 每天使用方式

1. 手機打開網站
2. 看 LINE 社群公告
3. 按「複製」
4. 貼到 LINE 社群
5. 再複製 Threads / IG 版本

## 修改文字

打開 `script.js`，修改 content 裡面的文字即可。
